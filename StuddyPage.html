Create and implement Objects and Methods

Objective: Implement native objects; create custom objects and custom properties for native objects using prototypes and functions; inherit from an object; implement native methods and create custom methods

Implementing Native objects. Native objects are by pre-defined Objects that come packaged with JavaScript. Some custom objects include, String(), Date(), Math(), JSON(), and regExp()

To create your own custom objects, you would use the following code in javascript. var person1 = {};. This block is semantically the same as var person1 = new Object();

With constructors and prototypes its important to note that these are used to decrease the amount of buffer overflow. If we had assigned a function to each constructor, that woudl make a method for for each constructor, all of them over lapping eachother in defintition each time a new contructor is created. So we use a single "prototype" to run against each specific object as we need too. Say, when we want to withdraw from an account, we run the "withdraw protoype" against the object like so accountInQuestion.withdraw(amount), the amountbeing any number you choose to input.
